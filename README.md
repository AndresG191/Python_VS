# ERP para Gestión Portuaria

## Descripción

Sistema desarrollado en Python para simular la administración de procesos dentro de un puerto, incluyendo inventario, empleados, auditoría y gestión de recursos mediante una base de datos relacional.


## Tecnologías

- Python
- MySQL
- SQL
- Docker
- Node.js

## Funcionalidades

- Gestión de inventario
- Gestión de empleados
- Auditoría
- CRUD
- Consultas SQL
- Relaciones 1:N
